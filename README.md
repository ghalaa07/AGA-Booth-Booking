# AGA Booth Booking

Interactive exhibition booth booking platform built with Next.js and React.

## Features

* View exhibition booths
* Select available booths
* Book booths
* View booth details
* Track booked and available booth counts

## Tech Stack

* Next.js
* React
* JavaScript
* GitHub
* Vercel
